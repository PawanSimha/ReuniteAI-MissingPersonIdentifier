# LinkedIn Project: ReuniteAI

**Project name**
ReuniteAI: AI-Powered Biometric Missing Person Identification & Recovery Platform

**Description**
Developed a mission-driven biometric identification system designed to reunite missing individuals with their families using state-of-the-art AI. The platform features an advanced face recognition engine built on `dlib`'s deep residual network, achieving 99.38% accuracy in matching unidentified persons against a centralized database.

The solution integrates a high-performance Python/Flask backend with a MongoDB NoSQL database for scalable record management. I designed a modern, intuitive SaaS-style interface that ensures accessibility for both citizens and administrators. By implementing NumPy-optimized matching pipelines and secure Bcrypt authentication, the project demonstrates a complete integration of computer vision engineering and secure web architecture for social impact.

**Top 5 skills used in this project**
1. AI & Computer Vision (Face Recognition & dlib)
2. Back-End Web Development (Python & Flask)
3. NoSQL Database Management (MongoDB)
4. Full-Stack Integration (Biometric Matching & Pipeline Orchestration)
5. UI/UX Design (Modern SaaS Aesthetic & Responsive Web Design)
